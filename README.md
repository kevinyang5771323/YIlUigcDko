# 基于深度学习的人脸识别系统 python742

## 项目概述

本项目是基于深度学习技术实现的人脸识别系统，支持多种人脸识别方式，并提供相应的人脸管理功能。

## 技术栈

- 开发环境：Pycharm
- 包管理器：Anaconda
- 编程语言：Python 3.8
- 人脸识别库：dlib 19.19.0
- 图形用户界面：PyQt5

## 功能模块

### 人脸识别

- 选择照片识别
  - 用户可以上传单人或多人人脸照片进行识别。
- 选择视频识别
  - 用户可以上传视频文件，系统将识别视频中出现的人脸。
- 摄像头实时检测识别
  - 实时捕捉摄像头画面，对检测到的人脸进行识别。

### 人脸管理

- 录入人脸
  - 用户可录入新的人脸信息，以进行后续的识别与管理。
- 管理人脸
  - 管理已录入的人脸信息，包括编辑和删除操作。

## 系统角色

- 用户：使用系统提供的各项功能，进行人脸识别与管理的操作。
